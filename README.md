# Tic Tac Toe C++ Game

A simple Tic Tac Toe game built in C++ using object-oriented programming.

## Features
- Player move input
- Computer-generated moves
- Win condition checking
- Score tracking
- Board reset functionality

## Concepts Used
- Classes and objects
- Inheritance
- Arrays
- Functions
- Random number generation
- Basic game logic

## Files
- `main.cpp` - Runs the game & holds function definitions
- `TicTacToe.h` - Class declarations
